# AI-Powered ECE Virtual Lab 🚀

An AI-powered Virtual Electronics Laboratory built using React, Node.js, Express, Gemini AI, and CircuitJS.

This platform helps Electronics and Communication Engineering students perform virtual experiments, understand circuits, generate reports, and prepare for viva questions interactively.

---

# 🌐 Live Demo

## Frontend
https://ai-ece-virtual-lab-mdrc.vercel.app/

## Backend
https://ai-ece-virtual-lab.onrender.com

---

# ✨ Features

## 🔬 Virtual Electronics Lab
- Interactive electronics circuit simulator
- Real-time virtual experimentation
- Circuit visualization using CircuitJS

## 🤖 AI ECE Assistant
Students can ask doubts related to:
- RC Circuits
- Filters
- Amplifiers
- Op-Amps
- Communication Systems
- Digital Electronics
- Waveforms
- Oscillators

## 📄 Smart Report Generator
- Generates lab reports automatically
- Helps students prepare observations and conclusions

## 🎤 Viva Question Generator
- AI-generated viva questions
- Helps students prepare for practical examinations

## 📊 Waveform Understanding
- Analyze and understand waveforms
- Learn electronics concepts interactively

## 🌍 Fully Deployed Full Stack Project
- Frontend deployed on Vercel
- Backend deployed on Render
